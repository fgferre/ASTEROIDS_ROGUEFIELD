## Balance Notes

**Why aggressive curve works**:

1. **Excitement builds** - Each upgrade feels MORE impactful than the last
2. **Goal to chase** - "I NEED Rank 5!" becomes the motivation
3. **Power fantasy** - Late game you feel like a GOD
4. **Replayability** - Starting fresh feels different knowing what's possible
5. **Risk/Reward** - Early game is hard, late game is your reward

**Doesn't break balance because**:

- Enemies scale with waves too
- More speed = harder to control (skill ceiling)
- Special abilities have cooldowns
- You still only have same HP/shields
